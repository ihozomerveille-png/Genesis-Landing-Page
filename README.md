 SaaS Landing Page Clone

 Description
This project is a responsive SaaS landing page built using **Tailwind CSS**. It demonstrates modern frontend layout techniques, utility-first styling, and responsive design. The page includes hero, features, testimonials, FAQ, and pricing sections, similar to a professional software product website.

 Features
- Hero section with headline, description, and call-to-action buttons
- Features section with cards displaying icons, titles, and descriptions
- Testimonials section with user quotes and images
- FAQ and Pricing sections
- Fully responsive design for mobile, tablet, and desktop
- Clean, modern design using Tailwind CSS utilities
- Semantic HTML5 structure

 Technologies Used
- HTML5
- Tailwind CSS (via CDN)
